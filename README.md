# vrrouter

访问"https://www.marzipano.net/tool/#"
将全景图片上传，下载成品，替换掉

- public/jsshader/data.js [用APP_DATA的内容替换RouterData的内容]
- public/jsshader/tiles

自定义public/jsshader/sense/场景名称[在RouterData中的scenes中每个场景的name部分，例如"id": "0-","name": "入校校门"]/index.html
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
